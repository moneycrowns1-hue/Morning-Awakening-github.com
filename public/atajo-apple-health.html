Morning Awakening

Conectar Apple Health

Para leer tus datos de sueño desde Apple Health, armamos un Atajo (Shortcut) de iOS que vos tenés que crear una vez en tu iPhone. Después, con un solo tap, se sincronizan las últimas noches con la app.

Crear el atajo

  1. Abrí la app Atajos (la que tiene el icono de dos flechas cruzadas azules) → tocá + arriba a la derecha.
  2. Tocá el nombre del atajo (arriba) y ponele exactamente: Morning Awakening Sync. Es importante el nombre exacto porque la app lo invoca así.
  3. Agregá la acción Find Health Samples → configurá Type: Sleep Analysis, Category: Asleep, Limit: 30, Sort by: End Date (Latest first).
  4. Agregá Repeat with each sobre el resultado anterior. Dentro del loop:
    • • Get Start Date of Repeat Item → guardar en variable s
    • • Get End Date of Repeat Item → guardar en variable e
    • • Format Number: {"start": {s en ms}, "end": {e en ms}} (usa Get numbers from input para obtener el timestamp en ms)
  5. Agregá Combine Text para juntar los objetos JSON separados por coma.
  6. Agregá Text con el JSON final: {"v":1,"exportedAt":{now en ms},"nights":[{lista}]}
  7. Agregá Base64 Encode sobre ese texto (modo URL-safe si está disponible).
  8. Agregá URL: https://tu-dominio.com/#hk= + el resultado anterior.
  9. Agregá Open URL. Esto abre Safari / la PWA con los datos en el hash.

Usarlo

  1. Volvés a Morning Awakening → en la pantalla de noche tocás el corazón ❤ arriba a la derecha → Conectar.
  2. Si es la primera vez, Apple te va a pedir permiso para leer Sueño desde Atajos. Tocás Permitir.
  3. El atajo corre, genera el JSON y te devuelve a la app. Ya vas a ver el corazón en verde y el estado “Sincronizado”.
Privacidad: los datos viajan solo en la URL del hash (#hk=…) que queda en tu navegador. No hacemos ningún request al servidor. Todo se guarda en localStorage de tu dispositivo.

Alternativa rápida (sin atajo)

Si todavía no querés armar el atajo, podés probar el puente pegando un JSON manualmente. Abrí esta URL en el navegador reemplazando <tu-json-base64>:

https://tu-dominio/#hk=<tu-json-base64>
